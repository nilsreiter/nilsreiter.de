Introduction

Goals

  • Create a gold standard for a couple of (relevant) texts
  • Allows for experimentation of simple hacks (dramas) as well as more complex systems
  • Be compatible to standard coreference as much as possible
    • Reuse of annotation guidelines (Naumann, 2007)
    • Potential integration of tools
    • Potential combination of annotated data (→ domain adaptation)
  • Apply the same/similar scheme to prose, drama, and non-literary texts

Expectations

  • Differences to vanilla coreference (= NLP coreference)
    • Text length: How is coreference established between (textually) distant mentions?
    • Different domains: Literature (prose & drama), speech

CRETA

Coreference Annotations

  • Status: About to start, two annotators per text (to begin)
  • Corpus
    • Goethe: Die Leiden des Jungen Werther (1787)
    • Perutz: Zwischen neun und neun (1918)
    • Rowlandson: Narrative of the Captivity and Restoration of Mrs. Mary Rowlandson (1682, english)
    • Bundestag: Parliamentary debates (1996-2015)
  • Annotations are done in WebAnno
    • Custom layer for directed edges (based on dependency layer)

Guidelines

  • Chains: Elements of a chain form an equivalence set
  • Singletons are not annotated
  • Non-nominals are included (as antecedents)
  • Deviations from vanilla coreference
    • No annotation of link type (e.g., anaphoric vs. coreferential vs. cataphoric)
      • Can be detected automatically
    • Discontinuous mentions (e.g., relative clauses) are not coreferential, but marked as discontinuous
      • Ich habe [1 die Sendungen] zur Post gebracht, [2 [3 die] hier schon lange rumliegen].
        • discont(2,1)

QuaDramA

Coreference Annotations

  • Annotation goal
    • Lessing: Miß Sara Sampson (Bürgerliches Trauerspiel)
    • Schiller: Die Räuber (Sturm und Drang)
  • Encoded in TEI-XML
  • Speakers are identified in a machine-readable way (Trilcke et al., 2015)
    • Each speaking character has a unique XML id, e.g. #sara, #einer_von_der_bande
  • Status: 2 acts done by three annotators in parallel, adjudication in progress
  • Things to talk about
    • Data representation & annotation setup
    • Interesting examples

Technical Requirements

  • Text structure is important for plays
    • Reducing to plain text entails a severe loss of information
  • Coreference information should be integratable into the TEI/XML:

Current Annotation Setup: Workflow

  1. Import TEI into UIMA pipeline (inline → standoff)
  2. Create coreference chains (of <speaker>-tags) using DKpro typesystem
  3. Export as XMI file
  4. Import XMI file into WebAnno
  5. Annotate in WebAnno!
  6. Export XMI from WebAnno
  7. Merge exported XMI into original (unannotated) XMI file (because WebAnno removes all non-DKpro annotations)
  8. Export as TEI/XML (inline annotations)

Current Annotation Setup: Assumptions/Restrictions

  • No overlapping annotations
    • <a>the <b>dog</a> barks</b> ←not proper XML
  • We can’t guarantee the order of multiple empty XML elements
    • <a/><b/> ↔︎ <b/><a/>

Annotation Issues & Findings

Generics

Der Wirt. […] Was liegt mir daran, ob ich es weiß, oder nicht, was Sie für eine Ursache hierher führt, und warum Sie bei mir im Verborgnen sein wollen? Ein Wirt nimmt sein Geld, und läßt seine Gäste machen, was ihnen gut dünkt. Waitwell hat mir zwar gesagt, […]

  • Annotators: Coreferent with #wirt
  • Guidelines: Ein Wirt starts a new coref chain
  • Decision: We follow the guidelines

Generics: Bonus Questions

  • Generic expressions are relatively frequent in this text
    • “[Ihr Mannspersonen] müßt doch selbst nicht wissen, was ihr wollt.”
    • “Mellefont besitzt alles, was uns [eine Mannsperson] gefährlich machen kann.”
    • Should generic expressions be co-referent (e.g., multiple references to the class man)?
  • Generic expressions are often understood as (partial) reference to non-generic entities
    • “Es war der Fehler [eines zärtlichen Mädchens], und [ihre] Flucht war die Wirkung ihrer Reue.”
    • It’s clear that they talk about #sara
    • If this is not co-reference, should it be annotated as another relation?

Zero-Quantification (?)

Mellefont. […] ich hatte noch keine verwahrlosete Tugend auf meiner Seele. Ich hatte noch keine Unschuld in ein unabsehliches Unglück gestürzt. Ich hatte noch keine Sara aus dem Hause eines geliebten Vaters entwendet, und sie gezwungen, einem Nichtswürdigen zu folgen, der auf keine Weise mehr sein eigen war.

(At utterance time, Mellefont has abducted Sara.)

  • Annotators: “keine Sara” = #sara
  • Nils: Similar to “Ich hatte Sara noch nicht entführt”
  • Sarah: Generic, because context: “kein Mädchen wie Sara”

Bridging vs. Coreference

Mellefont. Mit Unrecht tadelt sie die Verzögerung einer Zeremonie […].
[…]
Sara. Neue Freunde sollen die Zeugen unserer Verbindung sein? – Grausamer […]
[…]
Mellefont. Aber überlegen Sie denn nicht, Miß, daß unserer Verbindung hier diejenige Feier fehlen würde, die wir ihr zu geben schuldig sind?

  • “Verbindung” refers to both wedding (event) and marriage (state)
